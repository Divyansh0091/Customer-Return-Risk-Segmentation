# Customer Return Risk Analysis: A Statistical Approach

## 📝 Project Summary
This project employs **statistical hypothesis testing** to uncover actionable patterns in customer return behavior for an e-commerce enterprise. By utilizing a dataset of 10,000+ customer transactions, I performed a **Chi-Square Test of Independence** to analyze the correlation between customer segments and order return rates.

The analysis provides a data-driven strategy to mitigate return risks and optimize customer acquisition costs.

## 📊 Key Insights
* **Statistical Significance:** The analysis confirmed a strong dependency between customer segments and return behavior ($p < 0.05$).
* **Segment Performance:** Guest Users exhibit a disproportionately higher return rate (40%) compared to Premium Gold members (15%).
* **Actionable Strategy:** Recommended a shift from "Guest Checkout" to "Loyalty-Based Enrollment" to reduce logistical churn and improve long-term retention.

## 🛠️ Technical Stack
* **Language:** Python
* **Libraries:** Pandas (Data Manipulation), NumPy (Numerical Processing), SciPy (Statistical Inference), Seaborn/Matplotlib (Data Visualization)
* **Statistical Method:** Chi-Square Test of Independence

## 📈 Methodology
1. **Data Generation:** Simulated enterprise-grade categorical data using NumPy.
2. **Contingency Analysis:** Implemented `pd.crosstab` for matrix formation.
3. **Hypothesis Testing:** Executed Chi-Square analysis to validate correlations.
4. **Visual Storytelling:** Developed professional-grade stacked bar charts with grid-based visual anchors to communicate findings.
